# Social Network API

[![License: Unlicense](https://img.shields.io/badge/license-Unlicense-blue.svg)](http://unlicense.org/)

## Description

This is the backend for a social network api using MongoDB. Once set up you can use Insomnia or any other API client to test out the routes.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Credits](#credits)

## Installation

Step 1: Install all the packages:

```
npm i
```

Step 2: Seed the users

```
npm run seed
```

Step 3: Set up dev server

```
npm run dev
```

Step 4: Once the server is running, you can use the API client of your choice to test out the various functions.
